# ink-wasm

Rust → WebAssembly modules for browser-side cryptography, compression, markdown rendering, and fuzzy search. Zero JavaScript dependencies. Built for [seaofglass.ink](https://seaofglass.ink).

## Modules

### Compression (zstd)
```js
import init, { zstd_compress, zstd_decompress } from './ink_wasm.js';
await init('./ink_wasm_bg.wasm');

const compressed = zstd_compress(new TextEncoder().encode('hello'), 3);
const original = zstd_decompress(compressed); // Uint8Array
```

### Argon2id Key Derivation
```js
const key = argon2id_derive(
  password,     // Uint8Array
  salt,         // Uint8Array (16 bytes)
  65536,        // memory in KiB (64MB)
  3,            // iterations
  1,            // parallelism (WASM is single-threaded)
  32            // output length (256-bit key)
);
```

### Markdown → HTML
```js
const html = markdown_to_html('# Hello\n\n**bold** text');
// '<h1>Hello</h1>\n<p><strong>bold</strong> text</p>\n'
```

### Fuzzy Search
```js
const results = fuzzy_search(text, 'query', 20); // JSON string
// [{"line":42,"score":15,"text":"matched line content"}]
```

## Build

```bash
rustup target add wasm32-unknown-unknown
cargo install wasm-pack
wasm-pack build --target web --release
```

Output in `pkg/`:
- `ink_wasm_bg.wasm` — WASM binary (~548KB raw, ~184KB gzipped)
- `ink_wasm.js` — JS bindings (auto-generated by wasm-bindgen)

## Dependencies

| Crate | Purpose | Size contribution |
|---|---|---|
| `zstd` 0.13 | Zstandard compression | ~350KB |
| `argon2` 0.5 | Argon2id KDF | ~50KB |
| `pulldown-cmark` 0.12 | CommonMark parser | ~100KB |
| `wasm-bindgen` 0.2 | JS interop | ~10KB |

Fuzzy search is hand-rolled (no external crate).

## Performance

| Operation | Input | Time |
|---|---|---|
| zstd compress (level 3) | 10KB text | ~2ms |
| zstd decompress | 10KB compressed | ~1ms |
| argon2id (64MB, 3 iter) | password + salt | ~800ms |
| markdown → html | 5KB markdown | ~1ms |
| fuzzy search | 10K lines, 50 results | ~5ms |
