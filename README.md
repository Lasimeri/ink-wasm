# ink-wasm

Rust → WebAssembly modules for browser-side cryptography, compression, markdown rendering, fuzzy search, digital signatures, and secret sharing. Zero JavaScript dependencies. Built for [seaofglass.ink](https://seaofglass.ink).

## Modules

### Compression (zstd)
```js
import init, { zstd_compress, zstd_decompress } from './ink_wasm.js';
await init('./ink_wasm_bg.wasm');

const compressed = zstd_compress(new TextEncoder().encode('hello'), 3);
const original = zstd_decompress(compressed); // Uint8Array (10MB decompression limit)
```

### Argon2id Key Derivation
```js
const key = argon2id_derive(
  password,     // Uint8Array
  salt,         // Uint8Array (16 bytes)
  65536,        // memory in KiB (64MB)
  3,            // iterations
  1,            // parallelism (WASM is single-threaded)
  32            // output length (256-bit key)
);
```

### Markdown → HTML
```js
const html = markdown_to_html('# Hello\n\n**bold** ~~struck~~ text');
// '<h1>Hello</h1>\n<p><strong>bold</strong> <del>struck</del> text</p>\n'
```
Supports tables, strikethrough, and task lists via pulldown-cmark extensions.

### Fuzzy Search
```js
const results = fuzzy_search(text, 'query', 20); // JSON string
// [{"line":42,"score":15,"text":"matched line content"}]
```
Smith-Waterman-like scoring with bonuses for consecutive matches and word-boundary hits. Hand-rolled — no external crate.

### Ed25519 Signatures
```js
// Generate keypair — returns 64 bytes: [32-byte secret][32-byte public]
const keypair = ed25519_keygen();
const secret = keypair.slice(0, 32);
const pubkey = keypair.slice(32);

// Sign
const signature = ed25519_sign(secret, message); // 64-byte signature

// Verify
const valid = ed25519_verify(pubkey, message, signature); // boolean
```

### Shamir's Secret Sharing
```js
// Split a secret into 5 shares, requiring 3 to reconstruct
const packed = shamir_split(secret, 5, 3);
// packed format: [threshold][count][per share: 2-byte-BE length + data]

// Combine shares (pass packed blob containing >= threshold shares)
const recovered = shamir_combine(packed);
```
Uses `sss-rs` with verification enabled. Threshold must be >= 2.

## Build

```bash
rustup target add wasm32-unknown-unknown
cargo install wasm-pack
wasm-pack build --target web --release
```

Output in `pkg/`:
- `ink_wasm_bg.wasm` — WASM binary (~668KB raw, ~255KB gzipped)
- `ink_wasm.js` — JS bindings (auto-generated by wasm-bindgen)

## Dependencies

| Crate | Version | Purpose | Size contribution |
|---|---|---|---|
| `zstd` | 0.13 | Zstandard compression | ~350KB |
| `argon2` | 0.5 | Argon2id KDF | ~50KB |
| `pulldown-cmark` | 0.12 | CommonMark parser (tables, strikethrough, tasklists) | ~100KB |
| `ed25519-dalek` | 2 | Ed25519 signatures (keygen, sign, verify) | ~80KB |
| `rand_core` | 0.6 | Cryptographic RNG (OsRng for keygen) | ~5KB |
| `getrandom` | 0.2 | WASM-compatible entropy source (`js` feature) | ~5KB |
| `sss-rs` | 0.12 | Shamir's Secret Sharing (split/combine with verification) | ~30KB |
| `wasm-bindgen` | 0.2 | JS interop | ~10KB |

Fuzzy search is hand-rolled (no external crate).
